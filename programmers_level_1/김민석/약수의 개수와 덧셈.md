## 약수의 개수와 덧셈

```java
class Solution {
    public int solution(int left, int right) {
        int answer = 0;
        int count = 0;
        for(int i = left; i <= right; i++) {
            count = 0;
            for(int j = 1; j <= i; j++) {
                if(i % j == 0) {
                    count++;
                }
            }
            
            if(count % 2 == 0) {
                answer += i;
            } else if(count % 2 == 1) {
                answer -= i;
            }
        }
        
        return answer;
    }
}
```
